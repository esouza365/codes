Aula 4 de HTML

Nesta aula vamos aprender sobre Links, Imagens e Listas.

1️⃣ Links em HTML

Usamos a tag <a> (âncora) para criar links.

Estrutura:

<a href="URL">Texto do link</a>

Exemplo:

<a href="https://www.google.com">Ir para o Google</a>

Abrir em nova aba:

<a href="https://www.google.com" target="_blank">Abrir Google</a>

2️⃣ Imagens

Usamos a tag <img> para inserir imagens.

Estrutura:

<img src="caminho-da-imagem" alt="descrição">

Exemplo:

<img src="imagem.jpg" alt="Foto de exemplo">
